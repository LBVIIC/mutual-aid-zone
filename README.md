# 互助圈

## 说明

本项目源自于之前用 jQuery 写的校园互助平台，这次用 Vue 重构了一遍。

前端用的是 Vue3 + TypeScript + Vite + setup语法糖

后端用的是 Koa，代码见 [mutual-aid-zone-node](https://github.com/LBVIIC/mutual-aid-zone-node)

## 介绍

本项目为手机网页，实现了用户的登录注册，修改资料，发布接受任务，充值余额，兑换商品功能。
